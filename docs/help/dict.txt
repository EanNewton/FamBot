Following each command is alternate names that operate the same.
`!tr -DESTINATION -SOURCE PHRASE` Use Google to translate a PHRASE into another language. Language is the two letter Google code, e.g. en for English, jp for Japanese, etc.
-SOURCE is optional, if not provided the bot will attempt to autodetect it.
-DESTINATION is optional, if not provided the bot will attempt to translate into English.
Use `!tr langs` for a full list of language codes.
-- `!trans`
-- `!translate`

`!wolfram txt PHRASE` run PHRASE through the Wolfram Alpha engine and get a textual response
`!wolfram img PHRASE` run PHRASE trhough the Wolfram Alpha engine and get an image based response
